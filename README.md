# Landa - Website

Modern, professional one-page website for Landa - a Polish automotive chemistry warehouse and official distributor.

## Features

- **Responsive Design**: Mobile-first approach with full responsiveness across all devices
- **Modern UI**: Industrial-modern design with automotive theme elements
- **SEO Optimized**: Polish keywords, meta tags, and Schema.org structured data
- **Accessibility**: High contrast, keyboard navigation, ARIA labels
- **Performance**: Optimized loading, lazy loading, smooth animations
- **Interactive Elements**: 
  - Sticky navigation bar
  - Smooth scroll animations
  - Product filtering tabs
  - Contact form with validation
  - Floating action buttons (WhatsApp & Phone)
  - GDPR compliant cookie notice

## Sections

1. **Hero Section**: Company branding with professional car image and clear CTAs
2. **About Section**: Highlights 20 years of experience, excellent customer support, and company values
3. **Brands Section**: Display of 13 official partner brands with logo images
4. **Services Section**: Mieszalnia Lakierów, Linia Przemysłowa, Wsparcie Techniczne with service images
5. **Gallery Section**: Professional car painting and spraying images showcasing expertise
6. **Products Section**: Descriptive product categories (not e-commerce) with detailed information
7. **Contact Section**: Interactive map, contact information, and contact form

## Technologies

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Montserrat, Open Sans)
- Google Maps API (embedded)

## Setup

1. Open `index.html` in a web browser
2. All assets are self-contained (no build process required)

## Adding Images

The website is designed to display professional images. See `images/README.md` for detailed instructions on:
- Required image files and their locations
- Image specifications and recommended sizes
- Brand logo requirements
- Product and service image guidelines

**Important**: Add your images to the `images/` directory structure:
- `images/hero-car.jpg` - Hero section car image (shiny car)
- `images/warehouse-interior.jpg` - About section image
- `images/brands/*.png` - Brand logos (13 logos - can use publicly available brand logos)
- `images/services/*.jpg` - Service images (3 images, including woman painting)
- `images/gallery/*.jpg` - Gallery images (4 images with professional women painting/spraying cars)
- `images/products/*.jpg` - Product category images (3 images)

**Note**: The website is designed to showcase products and services, not as an e-commerce shop. Products are presented as descriptive categories with detailed information.

The website includes fallback behavior - it will display properly even if some images are missing.
