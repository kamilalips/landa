# Landa PL - Website

Modern, professional one-page website for Landa PL - a Polish automotive chemistry warehouse and official distributor.

## Features

- **Responsive Design**: Mobile-first approach with full responsiveness across all devices
- **Modern UI**: Industrial-modern design with automotive theme elements
- **SEO Optimized**: Polish keywords, meta tags, and Schema.org structured data
- **Accessibility**: High contrast, keyboard navigation, ARIA labels
- **Performance**: Optimized loading, lazy loading, smooth animations
- **Interactive Elements**: 
  - Sticky navigation bar
  - Smooth scroll animations
  - Product filtering tabs
  - Contact form with validation
  - Floating action buttons (WhatsApp & Phone)
  - GDPR compliant cookie notice

## Sections

1. **Hero Section**: Company branding with clear CTAs
2. **Brands Section**: Display of partner brands (Mirka, Novol, Matador, etc.)
3. **Services Section**: Mieszalnia Lakierów, Linia Przemysłowa, Wsparcie Techniczne
4. **Products Section**: Categorized product display with filtering
5. **Contact Section**: Interactive map, contact information, and contact form

## Technologies

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts (Montserrat, Open Sans)
- Google Maps API (embedded)

## Setup

1. Open `index.html` in a web browser
2. All assets are self-contained (no build process required)
